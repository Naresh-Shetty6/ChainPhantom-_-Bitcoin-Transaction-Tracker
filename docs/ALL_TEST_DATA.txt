QUICK USAGE GUIDE
===============================================================================

FOR WALLET ADDRESSES:
1. Go to: http://localhost:3000/wallet-monitor
2. Paste address in "Add Wallet to Monitor" input field
3. Click "+ Add Wallet"

OR

1. Direct URL: http://localhost:3000/address/{ADDRESS}
2. Replace {ADDRESS} with any address above


FOR TRANSACTION HASHES:
1. Direct URL: http://localhost:3000/transaction/{HASH}
2. Replace {HASH} with any transaction hash above


EXAMPLE URLS:
http://localhost:3000/address/mwm01AlertTriggerWallet1234567890abcdef
http://localhost:3000/transaction/tpd01fastsuccession1234567890abcdef1234567890abcdef
